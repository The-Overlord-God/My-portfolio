# My Building Journey

I am Md Althaf, a 18 year old self taught builder from Indore, India.

My journey started very early tbh. From childhood I was always curious how things work. I used to open electronics at home — mobiles, toys and whatever I could find 😅 sometimes even things I probably shoudnt open lol. I made simple cars using batteries and small motors. I was really fascinated by science experiments I saw on YouTube.

Once I even tried making hydrogen gas using electrolysis with a 9V adapter and a bottle (not the safest idea tbh). I also made a small perfume bottle rocket by spraying alcohol vapour and igniting it with a matchbox… later I switched to a piezo lighter which was kinda safer.

In 5th grade, I fixed an old CD player which stopped working. I opened it and found a loose solder connection and fixed it using a soldering iron. That was the first time I felt like “okay this is something I really like doing”.

During corona lockdown (I think 6th–7th grade), I got my first laptop. I joined a WhiteHat Jr trial class but before even attending it I started learning coding on my own 😂. I learned basic Java and made a simple bouncing ball game. Then I discovered Arduino… and that kinda changed everything.

I spent most of my free time learning sensors, if conditions, serial communication, motor drivers and all that stuff. In the school ATL lab, I built my first major project — a dual mode Arduino bluetooth car which could be controlled by both a mobile app and a physical joystick. I researched everything myself, soldered the circuit, wrote the code from scratch and kept improving it again and again. It even got featured in the school newspaper which felt crazy at that time.

I also did some experiments with high voltage stuff… like making a small tesla coil that produced visible sparks, and a hydrogen generator using electrolysis. I built multiple water pressure rockets during holi using tyre valves and air pumps and tested them with my friends (some failed badly lol).

In ATL lab I made a custom rocket and also a mars rover prototype for a NASA artemis inspired project. The teacher was actually shocked because I learned most of it from YouTube while being absent from class 😅. I was also part of a team trying to build a moon rover but it didnt really work out because of corona restrictions.

After 10th grade, I started learning Python more seriously. I made small useful tools like a photo duplicate finder, CSV maker and image sorter. I also participated in Codingal hackathons and reached around 4 kyu on Codewars in Python. I even started learning Arch Linux and deployed a public server using cloudflare tunnels (that part was kinda confusing at first ngl).

In the last few years, I focused more on 3D modeling. I use TinkerCAD for circuits and basic 3D designs like logic gates, 7 segment displays, rockets and rover models. I also learned Blender during holidays and made some renders and animations like the perfume bottle model and infinite cube loop.

Right now my main projects are:

Aura Glasses — my dream AI wearable with stereo distance measurement, object detection and gesture control
10x10x10 Tesseract LED Cube — a glowing cube with smoke and diffusion sheets for a kinda holographic 3D effect
Learning Android custom ROM development (still learning, its a bit tough tbh)

I build things because I enjoy learning. The word “impossible” actually motivates me more. I dont just learn something to make it… I make things to learn. Even if a project is hard I try not to leave it incomplete (okay sometimes it happens but yeah 😅). I repair my RC cars again and again, debug code for hours and just keep experimenting.
